Context Chung (Bắt buộc dán kèm cho mọi Task)
"Tôi đang làm app Quản lý sinh viên bằng Flutter.

Backend: Firebase Firestore (đã kết nối).

State: Provider (quản lý trong lib/providers/student_provider.dart).

Model: Student (trong lib/models/student.dart).

Quy tắc: Sử dụng Material 3, code sạch, tách nhỏ Widget."

👤 1. Prompt cho Hồ Đức Mạnh (Làm UI Card & About)
Nhiệm vụ: Tập trung vào thẩm mỹ, không cần lo về logic dữ liệu.

"Dựa trên Context Chung, hãy hoàn thiện file lib/widgets/student_card.dart.

Thiết kế UI cho StudentCard nhận vào một đối tượng Student.

Card cần có: Ảnh đại diện (CircleAvatar), Tên (Bold), MSSV, và một Badge hiển thị GPA.

GPA >= 3.2 màu xanh lá, GPA < 2.0 màu đỏ.

Sau đó, tạo file lib/views/about_screen.dart hiển thị danh sách 5 thành viên nhóm trong một ListView đẹp mắt, có nút quay lại."

👤 2. Prompt cho Nông Lan Anh (Làm Dashboard & Thống kê)
Nhiệm vụ: Xử lý dữ liệu từ Provider để vẽ biểu đồ.

"Dựa trên Context Chung, hãy tạo file lib/views/dashboard_screen.dart.

Sử dụng thư viện fl_chart để vẽ biểu đồ tròn (PieChart).

Dữ liệu lấy từ context.watch<StudentProvider>().students.

Phân loại sinh viên theo GPA: Xuất sắc (>=3.6), Giỏi (3.2-3.5), Khá (2.5-3.1), Trung bình (<2.5).

Hiển thị tổng số lượng sinh viên ở giữa biểu đồ."

👤 3. Prompt cho Nguyễn thành Dương (Làm Form Add/Edit)
Nhiệm vụ: Xử lý nhập liệu và gửi dữ liệu về Provider.

"Dựa trên Context Chung, hãy tạo file lib/views/add_student_screen.dart.

Tạo một Form gồm các TextFormField: Tên, MSSV, Lớp, Email, GPA.

Thêm Validation: GPA phải từ 0-4.0, Email phải đúng định dạng, không được bỏ trống các trường.

Khi nhấn nút 'Lưu', gọi hàm Provider.of<StudentProvider>(context, listen: false).addStudent(newStudent) và đóng màn hình bằng Navigator.pop."

👤 4. Prompt cho Trần Ngọc Lương (Logic nâng cao)
Nhiệm vụ: Mở rộng tính năng cho Provider và Service.

"Dựa trên Context Chung, hãy cập nhật lib/providers/student_provider.dart và lib/services/firebase_service.dart.

Viết thêm hàm updateStudent(Student student) để cập nhật thông tin sinh viên trên Firestore.

Viết hàm searchStudent(String query) để lọc danh sách _students theo Tên hoặc MSSV.

Đảm bảo các hàm này hoạt động Realtime và cập nhật UI thông qua notifyListeners()."
